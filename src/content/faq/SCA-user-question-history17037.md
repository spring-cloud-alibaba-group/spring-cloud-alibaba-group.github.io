 spring cloud 2023.0.1 对应springboot哪个版本？  

---------------
## 专家官方解答 ：

根据提供的知识内容，Spring Cloud 2023.0.1 对应的 Spring Boot 版本是 3.2.4。这意味着如果您想要使用 Spring Cloud 2023.0.1 版本，应当选择 Spring Boot 3.2.4 版本来确保两者之间的兼容性。
