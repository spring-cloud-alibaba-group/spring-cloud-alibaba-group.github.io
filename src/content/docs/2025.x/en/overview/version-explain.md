## 2025.x Branch

The 2025.x branch includes two major version lines, each adapted to different Spring Boot and Spring Cloud versions. Versions are listed from newest to oldest (the latest version is marked with \*):

### 2025.1.x

Adapted to Spring Boot 4.0.x and Spring Cloud 2025.1.x.

| Spring Cloud Alibaba Version | Spring Cloud Version | Spring Boot Version |
|------------------------------| -------------------- | ------------------- |
| 2025.1.0.0                   | 2025.1.0             | 4.0.0               |

### 2025.0.x

Adapted to Spring Boot 3.5.x and Spring Cloud 2025.0.x.

| Spring Cloud Alibaba Version | Spring Cloud Version | Spring Boot Version |
|------------------------------| -------------------- | ------------------- |
| 2025.0.0.0                   | 2025.0.0             | 3.5.0               |
